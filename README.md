# 📱 WhatsApp Message Scheduler

## 📖 Description
A desktop application built using Python and CustomTkinter to schedule WhatsApp messages.

---

## 🚀 Features

- 📱 Send WhatsApp Messages
- 💬 Custom Message
- 📅 Date Input
- ⏰ Time Input
- ✅ Input Validation
- 🎉 Success & Error Popup
- 🖥️ Modern GUI

---

## 🛠 Technologies Used

- Python
- CustomTkinter
- PyWhatKit
